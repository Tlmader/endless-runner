# endless-runner

v0.1

Endless runner game.

Author: Ted Mader

Genre: 3D Action

Tools Used:
Unreal Engine 4

Run the game:
Run EndlessRunner.exe

How to play:
* Press "A" to move left and "D" to move right. Press Space to jump!
* The runner goes faster the longer you last!

Objective:
* Avoid obstacles and collect coins!
